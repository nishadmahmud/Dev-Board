Question-1: What are the different ways to select an element in the DOM?

We can select elements using:

  • document.getElementById('id'): Selects an element by its unique ID.
  • document.querySelector('selector'): Selects the first element matching the CSS selector.
  • document.querySelectorAll('selector'): Selects all elements matching the CSS selector.
  • document.getElementsByClassName('class'): Selects elements by their class name.
  • document.getElementsByTagName('tag'): Selects elements by their tag name.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
Property Description
innerHTML This property allows you to get or set the HTML content inside an element. That means it includes all the HTML tags along with the text.
innerText This property gets or sets only the visible text inside an element. It ignores any hidden elements (like those with display: none;) and doesn't include any HTML tags. It also respects the styling of the page, meaning it only shows what a user would actually see on the screen.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we attach a single event listener to a parent element instead of individual listeners on child elements. It works because events triggered on child elements "bubble up" to their parent. Using this approach, you can handle events for multiple child elements efficiently without needing separate listeners for each.

Advantages:

  • Improves performance by reducing the number of event listeners.
  • Handles events for dynamically added or removed elements automatically.
  • Simplifies code by centralizing event handling logic.
Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event triggered on a child element propagates up through its ancestors in the DOM tree. This means that when an event occurs (e.g., a click), it is first handled by the innermost element (the target), and then successively by its parent elements until it reaches the root of the DOM tree.

For example, if we have a button inside a div, and both have click event listeners, clicking the button will trigger its event handler first, followed by the event handler of the div. This propagation can be stopped using methods like event.stopPropagation().

It enables a technique called event delegation, where we attach a single event listener to a parent element to handle events from its child elements. This improves performance and reduces memory usage, especially when dealing with dynamic or large numbers of child elements.

Question-5: How do you create, add, and remove elements using JavaScript?
  • Create: Use document.createElement('tagName') to create a new element.
  • Add: Use parentElement.appendChild(newElement) to add it to the DOM.
  • Remove: Use element.remove() or parentElement.removeChild(childElement) to remove an element.
